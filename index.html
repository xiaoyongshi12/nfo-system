国内稳定部署方案

国内替代部署方案

针对GitHub Pages在国内访问不稳定的问题，我们提供以下国内稳定部署方案：

腾讯云部署
Gitee部署
Vercel部署

1. 注册腾讯云账号

访问 腾讯云官网 注册账号并完成实名认证

新用户可享受免费试用资源

2. 开通静态网站托管

  1. 登录腾讯云控制台
  2. 搜索并进入"静态网站托管"服务
  3. 点击"立即使用"开通服务
  4. 创建新环境并选择"静态网站"

3. 上传网站文件

  1. 在本地将网站文件打包为ZIP格式（包含index.html和query.html）
  2. 在静态网站托管控制台点击"上传文件"
  3. 选择ZIP文件上传
  4. 上传完成后点击"开始部署"

4. 访问您的网站

部署完成后，系统会提供一个默认域名，格式为：

https://<your-id>.cos-website.<region>.myqcloud.com
腾讯云静态网站托管每月有免费额度，适合个人和小型项目使用
腾讯云部署文档

1. 注册Gitee账号

访问 Gitee官网 注册账号

国内用户建议使用手机号注册

2. 创建仓库并上传文件

  1. 点击右上角"+"创建新仓库
  2. 填写仓库名称，选择"公开"
  3. 上传index.html和query.html文件
  4. 点击"提交"按钮

3. 开启Gitee Pages

  1. 进入仓库，点击"服务" → "Gitee Pages"
  2. 选择部署分支（一般为master）
  3. 选择部署目录（一般为根目录）
  4. 点击"启动"按钮
注意：Gitee Pages需要手动更新，每次更新后需重新部署

4. 访问您的网站

部署完成后，系统会提供一个默认域名，格式为：

https://<username>.gitee.io/<repository-name>
Gitee Pages文档

1. 注册Vercel账号

访问 Vercel官网 使用GitHub账号注册

Vercel提供免费套餐，适合个人项目

2. 导入GitHub仓库

  1. 在Vercel控制台点击"Add New Project"
  2. 选择您的GitHub账号
  3. 选择包含网站代码的仓库
  4. 点击"Import"按钮

3. 配置部署设置

  1. 保留所有默认设置
  2. 确保框架预设为"Static Site"
  3. 输出目录留空
  4. 点击"Deploy"按钮

4. 访问您的网站

部署完成后，Vercel会提供一个免费域名：

https://<your-project>.vercel.app
提示：Vercel在国内访问速度较快，但建议绑定自定义域名以获得更好的访问体验
Vercel文档
部署后请确保使用HTTPS链接访问系统，移动设备才能正常扫描二维码。
